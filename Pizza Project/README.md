# Pizza Order System

A C# Windows Forms application that simulates a pizza ordering system. Users can customize their pizza by selecting the size, crust type, toppings, dining option, and quantity. The application automatically calculates the total price and displays a complete order summary.

## 📌 Features

- 🍕 Choose pizza size
  - Small
  - Medium
  - Large

- 🥖 Select crust type
  - Thin Crust
  - Thick Crust

- 🧀 Choose multiple toppings
  - Extra Cheese
  - Mushrooms
  - Tomatoes
  - Onion
  - Olives
  - Green Peppers

- 🍽️ Select dining option
  - Eat In
  - Take Out

- 🔢 Select pizza quantity

- 💲 Automatic price calculation

- 📋 Live order summary

- 🔄 Reset order

- ✅ Order confirmation using MessageBox

## 🛠️ Technologies Used

## ⚙️ How It Works

1. Choose the pizza size.
2. Select the crust type.
3. Add any desired toppings.
4. Choose whether to Eat In or Take Out.
5. Specify the quantity.
6. The application automatically updates:
   - Order Summary
   - Selected Size
   - Crust Type
   - Toppings
   - Dining Option
   - Total Price
7. Click **Order Pizza** to confirm the order.
