# RGB Color Picker

A simple C# Windows Forms application that demonstrates how to use **TrackBar** controls to create an interactive RGB color picker. The application's background color changes dynamically based on the Red, Green, and Blue values selected by the user.

## 📌 Features

- 🎨 Interactive RGB color picker
- 🔴 Adjust the Red color value
- 🟢 Adjust the Green color value
- 🔵 Adjust the Blue color value
- 🌈 Real-time background color update
- 📊 Display the current RGB values

## 🛠️ Technologies Used

- C#
- Windows Forms (.NET Framework)
- Visual Studio

 ⚙️ How It Works

1. Move any of the three TrackBars.
2. Each TrackBar controls one RGB color channel:
   - Red
   - Green
   - Blue
3. The application updates:
   - The RGB values.
   - The form's background color instantly.

## 🎯 Learning Objectives

This project helps beginners learn how to:

- Use TrackBar controls.
- Work with RGB color values.
- Create colors using `Color.FromArgb()`.
- Update the user interface dynamically.
- Handle Scroll events in Windows Forms.
