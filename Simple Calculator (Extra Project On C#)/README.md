# Simple Calculator

A C# Windows Forms calculator application that performs basic arithmetic operations along with additional mathematical functions such as square, square root, and modulus.

## 📌 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🟰 Modulus (%)
- 🟪 Square (x²)
- √ Square Root
- 🧹 Clear Screen
- ⚠️ Division by Zero Validation
- 📟 Dynamic Equation Display

## 🛠️ Technologies Used

- C#
- Windows Forms (.NET Framework)
- Visual Studio

## ⚙️ How It Works

1. Enter the first number.
2. Select an operation.
3. Enter the second number (if required).
4. Press the **=** button.
5. The application evaluates the expression and displays the result.

### Supported Operations

| Operation | Description |
|-----------|-------------|
| + | Addition |
| - | Subtraction |
| * | Multiplication |
| / | Division |
| % | Modulus |
| x² | Square |
| √x | Square Root |

- Handle button click events.

- Perform arithmetic calculations.
- Validate user input.
