# ComboBox Exercise

A simple C# Windows Forms application that demonstrates how to use a **ComboBox** to dynamically display different images and titles based on the selected item.

## 📌 Features

- Select an item from the ComboBox.
- Display the corresponding image.
- Update the title label automatically.
- Uses embedded project resources for images.

## 🛠️ Technologies Used

- C#
- Windows Forms (.NET Framework)
- Visual Studio

## 📂 Project Structure

- **ComboBox** for selecting an item.
- **PictureBox** to display the selected image.
- **Label** to display the selected item's name.
- **Resources** folder containing all images.

## 🖼️ Available Options

- 👦 Boy
- 👧 Girl
- 📚 Book
- 🖊️ Pen

## 🚀 How It Works

1. Run the application.
2. Select an item from the ComboBox.
3. The application updates:
   - The displayed image.
   - The title label.
