# C# Windows Forms Projects

Welcome to my **C# Windows Forms Projects** repository.

This repository contains the projects I developed while learning **C#** and **Windows Forms**. Each project focuses on practicing different programming concepts, controls, and application logic.

---

## Technologies Used

- C#
- .NET Framework
- Windows Forms
- Visual Studio

---

## Projects

| Project | Description |
|---------|-------------|
| 🎮 XO Game | A two-player Tic-Tac-Toe game with win detection, draw detection, and restart functionality. |
| 🍕 Pizza Project | Pizza ordering application using RadioButtons, CheckBoxes, GroupBoxes, and order summary. |
| ⏱️ Pomodoro Timer | Countdown timer with start, stop, reset, and time selection features. |
| 🧮 Simple Calculator | Basic calculator supporting common arithmetic operations. |
| 🎨 Drawing | Practice project for drawing graphics using GDI+ in Windows Forms. |
| 📋 ComboBox Exercise | Learning how to work with ComboBox controls. |
| 🖼️ PictureBox Exercise | Working with images and PictureBox controls. |
| 🌳 TreeView & ImageList | Practice using TreeView and ImageList controls. |
| 📑 MenuStrip Sample | Example of creating menus using MenuStrip. |
| 🪟 MDI Sample | Practice creating Multiple Document Interface (MDI) applications. |
| 🎚️ TrackBar Exercise | Working with TrackBar controls and value changes. |
| 🎛️ Mixed Controls Practice | Practice combining multiple Windows Forms controls in one application. |

---
