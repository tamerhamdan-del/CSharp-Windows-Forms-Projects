# Picture Box Exercise

A simple C# Windows Forms application that demonstrates how to use **PictureBox** and **RadioButton** controls to display different images dynamically based on the user's selection.

## 📌 Features

- Display images using a PictureBox.
- Select an image with RadioButtons.
- Update the title label automatically.
- Uses embedded project resources (`Resources.resx`).
- Beginner-friendly Windows Forms example.

## 🛠️ Technologies Used

- C#
- Windows Forms (.NET Framework)
- Visual Studio

## ⚙️ How It Works

1. Run the application.
2. Select one of the available RadioButtons.
3. The application automatically:
   - Displays the corresponding image.
   - Updates the title label with the selected item's name.

## 🎯 Learning Objectives

This project helps beginners learn how to:

- Display images using the PictureBox control.
- Handle RadioButton selection events.
- Use the `Tag` property to reduce hardcoded text.
- Update the user interface dynamically.
- Work with embedded resources in Windows Forms.
