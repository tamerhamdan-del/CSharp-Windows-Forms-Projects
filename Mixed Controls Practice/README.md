# Windows Forms Controls Collection

A comprehensive C# Windows Forms project that demonstrates the usage of many common Windows Forms controls and components. The project serves as a learning reference, with each feature implemented in a separate form.

## 📌 Features

This project contains **23 different Windows Forms examples**, including:

- Forms (Show & ShowDialog)
- MessageBox
- CheckBox
- RadioButton
- TextBox
- MaskedTextBox
- ComboBox
- LinkLabel
- CheckedListBox
- DateTimePicker
- MonthCalendar
- Timer
- NotifyIcon
- TreeView
- ImageList
- ProgressBar
- ListView
- ErrorProvider
- TrackBar
- NumericUpDown
- TabControl
- GroupBox & Panel
- Common Dialogs

## 🛠️ Technologies Used

- C#
- Windows Forms (.NET Framework)
- Visual Studio

## 📷 Preview

The main window provides quick access to **23 Windows Forms examples**. Each button opens a separate form demonstrating a specific control or component.

## ⚙️ How It Works

1. Launch the application.
2. Click any button on the main form.
3. A new form opens, demonstrating a specific Windows Forms control or feature.
4. Explore each example independently to understand its behavior and implementation.

This project helps beginners learn how to:

- Work with the most commonly used Windows Forms controls.
- Handle user events.
- Open and manage multiple forms.
- Build interactive desktop applications.
- Understand the fundamentals of event-driven programming in C#.
