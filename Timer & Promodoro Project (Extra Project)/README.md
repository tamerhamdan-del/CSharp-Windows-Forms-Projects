# Pomodoro Timer

A simple C# Windows Forms application that implements a basic **Pomodoro Timer**. Users can select a countdown duration, start the timer, and monitor the remaining time through a clean and easy-to-use interface.

## 📌 Features

- ⏱️ Countdown timer
- 📋 Select timer duration from a ComboBox
- ▶️ Start timer with one click
- ⏳ Live minutes and seconds display
- 🛑 Automatically stops when the countdown reaches zero
- 🎯 Beginner-friendly implementation

## 🛠️ Technologies Used

- C#
- Windows Forms (.NET Framework)
- Visual Studio

## ⚙️ How It Works

1. Select the desired countdown time from the ComboBox.
2. Click the **Start** button.
3. The timer begins counting down minute by minute and second by second.
4. When the timer reaches **00:00**, it automatically stops.
